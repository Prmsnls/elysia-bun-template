# AI Agent Guide

This project is a Bun + Elysia backend template.

## Quick Start
- **Build & Run**: `bun run docker:up`
- **Local Dev**: `bun dev`
- **Database**: Turso (libSQL) + Drizzle ORM. See `drizzle.config.ts`.
